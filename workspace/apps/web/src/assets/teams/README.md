# Cyprus Football Team Logos

This directory should contain the team logos (shields/badges) for Cyprus U19 League teams.

## Required Logo Files

Place the following logo files in this directory (PNG format, recommended size: 256x256px or larger):

- `anorthosis.png` - Anorthosis Famagusta U19
- `pafos.png` - Pafos FC U19
- `olympiakos.png` - Olympiakos Nicosia U19
- `omonoia.png` - Omonoia FC U19
- `aek.png` - AEK Larnaca U19
- `apoel.png` - APOEL Nicosia U19
- `ael.png` - AEL Limassol U19
- `karmiotissa.png` - Karmiotissa Pano Polemidion U19
- `aris.png` - Aris Limassol U19
- `nea-salamina.png` - Nea Salamina Famagusta U19
- `ayia-napa.png` - Ayia Napa U19

## Where to Get Logos

### Free Resources:

1. **Football-Logos.cc**
   - URL: https://football-logos.cc/cyprus/
   - Free downloads in various sizes and formats

2. **Wikimedia Commons**
   - URL: https://commons.wikimedia.org/wiki/Category:SVG_association_football_logos_of_Cyprus
   - High-quality SVG files

3. **SportsLogos.net**
   - URL: https://www.sportslogos.net/logos/list_by_team/1579/Cyprus-Logos/
   - Historical logos and uniforms

### Important Notes:

- **Usage Rights**: Ensure you comply with licensing agreements. Logos are property of their respective clubs.
- **Format**: PNG format recommended (transparent background preferred)
- **Size**: 256x256px minimum for good quality display
- **Naming**: Use lowercase with hyphens as shown above
